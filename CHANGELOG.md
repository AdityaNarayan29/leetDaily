# Changelog

All notable changes to LeetDaily will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.1] - 2025-12-14

### Added
- Yesterday's problem quick access
- Company tags for problems
- Smart notification reminders
- Improved branding and UI polish

### Changed
- Updated difficulty and acceptance rate display
- Enhanced styling consistency across themes
- Minor UI refinements

### Fixed
- Null checks for DOM elements to prevent errors
- CSS property fixes for better cross-browser support

## [1.1.0] - 2025-12-01

### Added
- LeetCode user data integration
- Streak tracking and display
- Gradient theme support

### Changed
- Updated manifest with improved description
- Enhanced theme variables for better styling

## [1.0.0] - 2025-11-15

### Added
- Initial release
- Daily LeetCode challenge display
- Problem difficulty and acceptance rate
- Topic tags display
- One-click access to problems
- Local storage for progress tracking
